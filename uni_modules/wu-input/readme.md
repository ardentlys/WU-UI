## wu-input 输入框

> **组件名：wu-input**

此组件为一个输入框，默认没有边框和样式，是专门为配合表单组件 [wu-form](https://ext.dcloud.net.cn/plugin?name=wu-form) 而设计的，利用它可以快速实现表单验证，输入内容，下拉选择等功能。
