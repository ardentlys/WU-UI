## wu-overlay 遮罩层

> **组件名：wu-overlay**

创建一个遮罩层，用于强调特定的页面元素，并阻止用户对遮罩下层的内容进行操作，一般用于弹窗场景，wu-ui 所有相关弹窗组件都是使用的该组件。
